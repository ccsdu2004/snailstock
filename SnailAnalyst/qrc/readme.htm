蜗牛股票量化分析软件是一款专业的股票策略评估软件

使用本软件可以快速评估一套炒股策略的成功率，最大回撤，某一时段的交易次数以及交易数据

当前，基础版本的蜗牛量化分析软件不收费，这意味着您可以随意使用本软件而无需取得作者的许可

 

当前，本软件的功能有

  1. 设定股票买点，包含但不限于OHLC，MA，VOLMA，BOLL，KDJ，DMI等技术指标以及K线形态
  2. 设定股票卖点，指标同上
  3. 设定单日最大允许买入股票数量
  4. 单日买入股票的优先级排序 - 当前基于成交量
  5. 股票策略结果显示股票回溯
  6. 免费版本仅提供480股股票数据
  7. 其他
